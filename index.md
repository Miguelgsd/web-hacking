# 🛡️ Web Pentesting

Bem-vindo ao meu repositório de estudos focado em Segurança de Aplicações Web. Postarei aqui todo o meu estudo na área de Hacking em sites.

---

## 🏗️ Metodologia de Teste
Minha abordagem segue os padrões do **OWASP WSTG** (Web Security Testing Guide):

1. **Reconhecimento & Footprinting** (Subdomínios, Tech Stack)
2. **Enumeração de Vulnerabilidades** (Fuzzing, Scanners)
3. **Exploração** (SQLi, XSS, IDOR, LFI/RFI)
4. **Pós-Exploração** (Escalação de privilégios, Exfiltração)
