Other > Remote File Inclusion

Utilizando o parâmetro "?page=", é possível incluir outra página dentro do site vulnerável.
Exemplo: 
	?page=http://www.google.com

Isso serve para qualquer página que permita tal funcionalidade
Quando se trata de uma inclusão de outra página (remotamente), se trata de um Remote File Inclusion (RFI)
